# UI Specification

## 概要

単一画面のデスクトップアプリとする。

目的は、CLI を知らない状態でも音声ファイルから字幕ファイルを生成できるようにすることである。

## 画面構成

```text
┌──────────────────────────────────┐
│ Whisper Subtitle                 │
├──────────────────────────────────┤
│ 音声ファイル                      │
│ [ /path/to/voice.wav      ] [選択] │
│                                  │
│ 出力先フォルダ                    │
│ [ /path/to/output         ] [選択] │
│                                  │
│ 出力ファイル名                    │
│ [ haikei001              ]        │
│                                  │
│ モデル                            │
│ [ large-v3 ▼             ]        │
│                                  │
│ 出力形式                          │
│ [ srt ▼                  ]        │
│                                  │
│ 言語                              │
│ [ ja ▼                   ]        │
│                                  │
│ [ 字幕生成 ]                      │
│                                  │
│ ログ                              │
│ Loading model...                  │
│ Transcribing audio...             │
└──────────────────────────────────┘
```

## UI 方針

- ミニマル
- ダークモード対応
- 余白を広めに取る
- 入力欄は見やすくする
- 字幕生成ボタンは明確に目立たせる
- 実行中はボタンを disabled にする
- 実行中ステータスを表示する

## 入力項目

### 音声ファイル

- ファイル選択ボタンを配置する。
- ドラッグ&ドロップに対応する。
- 選択済みファイルパスを表示する。

### 出力先フォルダ

- フォルダ選択ボタンを配置する。
- 選択済みフォルダパスを表示する。

### 出力ファイル名

- テキスト入力。
- 拡張子は入力させない。
- 空文字不可。

### モデル

セレクトボックス。

選択肢:

- tiny
- base
- small
- medium
- large-v3

初期値:

```text
large-v3
```

### 出力形式

セレクトボックス。

選択肢:

- srt
- vtt
- txt

初期値:

```text
srt
```

### 言語

セレクトボックス。

選択肢:

- ja
- en

初期値:

```text
ja
```

## 字幕生成ボタン

以下の場合は disabled にする。

- 音声ファイル未選択
- 出力先未選択
- タイトル未入力
- 実行中

クリック時に Main Process 経由で `subtitle.py` を実行する。

## ログ表示

- stdout と stderr を時系列で表示する。
- stderr は視認しやすくする。
- 自動スクロールする。
- 実行開始時に前回ログをクリアする。

## 成功表示

成功時は以下を表示する。

```text
Subtitle generated successfully.
```

可能であれば、生成されたファイルパスを表示する。

## 失敗表示

失敗時は以下を表示する。

```text
Subtitle generation failed.
```

stderr または例外メッセージも表示する。

## 設定保存

以下を localStorage に保存する。

- model
- format
- language
- output directory

次回起動時に復元する。
